## Flow 1: First Launch (Первый запуск)

**Задача:** Провести пользователя от установки до первой сессии растяжки за < 3 минут.  
**Фичи:** F1 (Onboarding Quiz), F10 (Profile)

### Happy Path

```
[App Store Install]
    → Splash Screen (логотип, 1–2 сек)
    → Welcome Screen (value prop + "Начать")
    → Quiz Step 1: Зона боли (мульти-выбор: шея, спина, глаза, запястья, всё)
    → Quiz Step 2: Частота боли (иногда / часто / каждый день)
    → Quiz Step 3: Social Proof + Тип работы (дом / офис / гибрид)
    → Quiz Step 4: Цель + Часы за столом
    → Labor Illusion (3–5 сек: "Подбираем упражнения...")
    → Персональный план (AHA-момент: "14 упражнений для вас")
    → Reminder Setup (утро / обед / вечер / каждые 2 часа)
        → [Системный запрос push-разрешения]
    → Paywall (timeline + персонализация + 7-day trial)
        → [Пользователь начинает trial ИЛИ нажимает "Позже"]
    → Home Screen
    → [Первая сессия растяжки — автоматическая рекомендация по зоне боли]
```

### Error States

| Ошибка | Поведение |
|--------|-----------|
| Нет интернета при запуске | Онбординг работает оффлайн (данные сохраняются локально). Paywall показывает "Нет соединения — попробуйте позже". Пользователь попадает на Home с бесплатным контентом |
| Пользователь отклоняет push | Экран "Напоминания помогают 87% пользователей" → кнопка "Настроить позже" → переход к paywall. Повторный запрос через 3 дня после первой завершённой сессии |
| Crash / kill app во время онбординга | Прогресс quiz сохраняется. При повторном запуске — продолжение с последнего шага |

### Edge Cases

| Сценарий | Поведение |
|----------|-----------|
| Пропуск quiz (кнопка "Skip") | Default-профиль: шея + спина, средняя частота, удалёнка. Home с generic рекомендациями. Баннер "Настроить профиль" на Home |
| Повторная установка (reinstall) | Если был аккаунт — предложение войти. Если нет — полный онбординг заново |
| Уже есть аккаунт (sign in) | Кнопка "Уже есть аккаунт?" на Welcome → Sign In → Home (данные восстановлены) |

## Flow 2: Core Action (Сессия растяжки)

**Задача:** От ощущения боли до завершённого упражнения за 2 нажатия.  
**Фичи:** F2 (Body-Part Targeting), F3 (Video Player), F6 (Eye Exercises)

### Happy Path

```
[Пользователь чувствует боль в шее]
    → Открывает приложение → Home Screen
    → Нажимает зону "Шея" на Body Map
    → Routine Preview (3 упражнения, ~3 мин)
        → Кнопка "Начать"
    → Exercise Player
        → Видео упражнение 1 (play/pause, таймер, подсказки)
        → Автопереход → Упражнение 2
        → Автопереход → Упражнение 3
    → Session Complete (поздравление + статистика)
        → Стрик обновлён
        → Pain check-in (опционально): "Как шея сейчас?" (1–10)
    → Home Screen (обновлённый стрик)
```

### Happy Path (Eye Break — короткий формат)

```
[Push-уведомление: "Время для глаз — 30 сек"]
    → Нажатие на push → Eye Exercise Screen
    → Анимированное упражнение (30 сек, без звука)
    → "Готово! Глаза отдохнули" → Закрытие
```

### Error States

| Ошибка | Поведение |
|--------|-----------|
| Видео не загружается | Fallback: текстовая инструкция + статичные иллюстрации + кнопка "Повторить загрузку" |
| Нет интернета | Кешированные видео работают. Если видео не кешировано: "Нет соединения. Загрузите видео заранее в настройках" |
| Входящий звонок во время сессии | Автопауза. При возврате — "Продолжить с места остановки?" |
| Приложение уходит в background | Автопауза. Таймер останавливается |

### Edge Cases

| Сценарий | Поведение |
|----------|-----------|
| Все упражнения зоны уже сделаны сегодня | "Отличная работа! Попробуйте другую зону" + предложение спина/глаза |
| Пользователь пропускает все упражнения (Skip) | Сессия не засчитывается в стрик. "Хотите попробовать другую рутину?" |
| Premium-контент для free-пользователя | Замок на упражнении → нажатие → мини-paywall: "Откройте 50+ упражнений" |
| Первая сессия (ever) | Дополнительные подсказки: "Нажмите play", "Повторяйте за инструктором" |

## Flow 3: Return Visit (Повторный визит)

**Задача:** Вернувшийся пользователь сразу видит ценность и начинает сессию.  
**Фичи:** F2, F5 (Reminders), F8 (Streaks), F9 (Pain Tracking)

### Happy Path (Через push-напоминание)

```
[Push: "Растяжка шеи — 3 мин. Твой стрик: 5 дней 🔥"]
    → Нажатие → Routine Preview (персональная рутина)
    → "Начать" → Exercise Player → Session Complete
    → Стрик: 6 дней
    → Home Screen
```

### Happy Path (Самостоятельное открытие)

```
[Пользователь открывает приложение]
    → Home Screen:
        ├── Streak widget: "5 дней подряд!"
        ├── Рекомендованная рутина: "Утренняя шея — 3 мин"
        ├── Eye Break: "Глаза: следующий через 18 мин"
        ├── Недельный календарь активности
        └── Pain check-in баннер (если не заполнен сегодня)
    → Выбирает рекомендованную рутину ИЛИ зону на Body Map
    → Exercise Player → Session Complete
```

### Error States

| Ошибка | Поведение |
|--------|-----------|
| Push не доставлен (DnD / отключены) | Следующий запуск: "Вы пропустили разминку вчера. Сделаем сейчас?" |
| Нет интернета | Home загружается из кеша. Кешированные рутины доступны. Баннер "Оффлайн-режим" |

### Edge Cases

| Сценарий | Поведение |
|----------|-----------|
| Вернулся через 7+ дней (re-engagement) | Мягкий экран: "С возвращением! Стрик сброшен, но прогресс сохранён. Начнём заново?" Нет guilt-messaging |
| Стрик под угрозой (вечер, ещё нет сессии) | Вечерний push (мягкий): "Ещё есть время сохранить стрик — 2 мин" |
| Grace day использован | Баннер: "Вчера был grace day. Сегодня важно не пропустить!" |
| Вернулся через 30+ дней | Re-onboarding lite: "Что-то изменилось? Обновите зоны боли" → обновление профиля → Home |

## Flow 4: Upgrade Flow (Конверсия в платную подписку)

**Задача:** Естественный переход от бесплатного использования к подписке через демонстрацию ценности.  
**Фичи:** F7 (Sciatica Program), F4 (Library), Paywall

### Happy Path (После онбординга — основная точка)

```
[Персональный план показан (AHA)]
    → Paywall Screen:
        ├── Персонализация: "Марина, ваша программа для шеи готова"
        ├── Timeline: "Сегодня → День 5: напоминание → День 7: списание"
        ├── Bullet-список Pro: персональные рутины, все зоны, программы, без рекламы
        ├── Ценовое якорение: $4.99/мес vs $2.49/мес (годовой)
        ├── Social proof: "4.8★ — 2,400+ отзывов"
        └── Кнопка "Начать бесплатный 7-дневный trial"
    → [Подтверждение App Store / Google Play]
    → Success: "Trial активирован! 7 дней бесплатно"
    → Home Screen (premium unlocked)
```

### Happy Path (Feature-gate — повторный показ)

```
[Пользователь на Home → Programs → Sciatica Program]
    → Экран программы: 2 упражнения доступны, остальные с замком
    → Нажатие на заблокированное → Mini-Paywall:
        ├── "Откройте программу для ишиаса"
        ├── Персонализация по текущему прогрессу
        └── "7 дней бесплатно"
    → [Подтверждение] → Success → Программа разблокирована
```

### Happy Path (После value moment)

```
[Session Complete → "Вы сделали 7 дней подряд!"]
    → Milestone celebration
    → "Хотите ещё больше? Pro открывает 50+ упражнений"
    → Paywall (контекстный) → [Подтверждение] → Success
```

### Error States

| Ошибка | Поведение |
|--------|-----------|
| Ошибка оплаты (карта отклонена) | "Не удалось обработать платёж. Проверьте данные карты в [App Store / Google Play]" + кнопка "Попробовать снова" |
| Нет интернета при покупке | "Нет соединения. Подключитесь к интернету для оформления подписки" |
| Пользователь закрывает paywall | Переход на Home с бесплатным контентом. Следующий показ paywall — через feature-gate или через 3 дня |
| Restore purchase не работает | "Не найдена активная подписка. Если вы уверены, что подписаны — напишите в поддержку" |

### Edge Cases

| Сценарий | Поведение |
|----------|-----------|
| Trial закончился, не конвертировался | Push Day 6: "Завтра заканчивается trial. Сохраните доступ!". Push Day 7: "Trial закончился. Вернитесь к Pro". Ограниченный контент (3 упражнения/зона). Paywall при попытке открыть premium |
| Уже подписан (restore) | Кнопка "Восстановить покупку" на paywall → проверка → "Подписка восстановлена!" |
| Пользователь из страны без поддержки оплаты | Информационный экран + ссылка на веб-версию оплаты |

## Flow 5: Share / Social Flow (Шеринг прогресса)

**Задача:** Пользователь делится достижением, привлекая новых пользователей.  
**Фичи:** F8 (Streaks & Gamification)

### Happy Path (Milestone achievement)

```
[Session Complete → Milestone: "7 дней подряд!"]
    → Celebration Screen:
        ├── Анимация + бейдж "7-Day Warrior"
        ├── Статистика: "21 мин растяжки, 3 зоны"
        └── Кнопка "Поделиться"
    → Share Sheet:
        ├── Автоматически сгенерированная карточка:
        │   "Я сделал 7 дней растяжек за столом 💪 DeskCare"
        │   + бейдж + стрик + ссылка на приложение
        ├── Каналы: Instagram Stories, WhatsApp, Telegram, Copy Link
        └── [Пользователь выбирает канал]
    → Шеринг завершён
    → "Спасибо! Вы помогаете коллегам двигаться больше"
```

### Happy Path (Weekly Summary)

```
[Воскресенье вечер → Push: "Ваш итог недели готов"]
    → Открытие → Progress Screen:
        ├── "На этой неделе: 5 сессий, 15 мин, шея + спина"
        ├── Pain trend: "Боль в шее: 7 → 5 (↓ 29%)"
        └── Кнопка "Поделиться итогом"
    → Share Sheet (карточка с недельной статистикой)
```

### Error States

| Ошибка | Поведение |
|--------|-----------|
| Share sheet не открывается | Fallback: "Скопировать ссылку" → текст в буфер обмена |
| Нет установленных соцсетей | Показать: Copy Link, Email, SMS |

### Edge Cases

| Сценарий | Поведение |
|----------|-----------|
| Нет достижений для шеринга | Кнопка "Поделиться" неактивна до первого milestone (3 дня) |
| Free-пользователь шерит | Шеринг доступен всем. Карточка содержит deep link. Получатель видит App Store page |
| Получатель переходит по ссылке | Deep link → App Store → Install → Onboarding (рефереру не начисляется ничего в MVP) |

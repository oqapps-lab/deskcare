## Детали экранов

### 1. Onboarding Flow

#### 1.1 Splash Screen
- **Назначение:** Брендинг, загрузка ресурсов
- **Откуда:** Первый запуск приложения
- **Куда:** Welcome Screen (1.2)
- **Элементы:** Логотип DeskCare, индикатор загрузки (subtle)
- **Длительность:** 1–2 сек

#### 1.2 Welcome Screen
- **Назначение:** Захватить внимание, объяснить value prop за 5 секунд
- **Откуда:** Splash (1.1) или повторный запуск (если онбординг не завершён)
- **Куда:** Quiz Step 1 (1.3) или Sign In (2.1)
- **Элементы:**
  - Иллюстрация: человек за столом делает растяжку
  - Заголовок: "2 минуты в день — шея перестанет болеть"
  - Подзаголовок: "Микро-растяжки прямо за рабочим столом"
  - CTA: "Начать" (primary button)
  - Ссылка: "Уже есть аккаунт? Войти"

#### 1.3 Quiz: Зона боли
- **Назначение:** Главный персонализирующий вопрос (определяет UVP)
- **Откуда:** Welcome (1.2)
- **Куда:** Quiz Step 2 (1.4)
- **Элементы:**
  - Прогресс-бар: 20%
  - Заголовок: "Что беспокоит больше всего?"
  - Мульти-выбор (карточки с иконками): Шея, Спина/Поясница, Глаза, Запястья, Всё тело
  - CTA: "Далее"
  - Ссылка: "Пропустить" (мелкий текст)

#### 1.4 Quiz: Частота боли
- **Назначение:** Определение интенсивности для подбора программы
- **Откуда:** Quiz Step 1 (1.3)
- **Куда:** Quiz Step 3 (1.5)
- **Элементы:**
  - Прогресс-бар: 40%
  - Заголовок: "Как часто это беспокоит?"
  - Single-select: Иногда / Несколько раз в неделю / Каждый день
  - CTA: "Далее"

#### 1.5 Quiz: Social Proof + Тип работы
- **Назначение:** Снизить drop-off через social proof + собрать контекст работы
- **Откуда:** Quiz Step 2 (1.4)
- **Куда:** Quiz Step 4 (1.6)
- **Элементы:**
  - Прогресс-бар: 60%
  - Social proof блок: "87% пользователей с болью в шее отмечают улучшение за 14 дней"
  - Заголовок: "Где вы работаете?"
  - Single-select: Дома / В офисе / Гибрид
  - CTA: "Далее"

#### 1.6 Quiz: Цель + Часы за столом
- **Назначение:** Определение мотивации и частоты напоминаний
- **Откуда:** Quiz Step 3 (1.5)
- **Куда:** Labor Illusion (1.7)
- **Элементы:**
  - Прогресс-бар: 80%
  - Заголовок: "Какая ваша цель?"
  - Single-select: Убрать боль / Профилактика / Больше энергии
  - Второй вопрос: "Сколько часов за столом?"
  - Single-select: 4–6 / 6–8 / 8+
  - CTA: "Готово"

#### 1.7 Labor Illusion
- **Назначение:** Повышение perceived value через "работу" системы
- **Откуда:** Quiz Step 4 (1.6)
- **Куда:** Персональный план (1.8)
- **Элементы:**
  - Анимация загрузки (прогресс 0–100%)
  - Текстовые этапы: "Подбираем упражнения..." → "Рассчитываем расписание..." → "Готовим программу..."
  - Длительность: 3–5 сек
  - Автоматический переход

#### 1.8 Персональный план (AHA-момент)
- **Назначение:** Показать ценность ДО paywall — ключевой экран конверсии
- **Откуда:** Labor Illusion (1.7)
- **Куда:** Reminder Setup (1.9)
- **Элементы:**
  - Заголовок: "Ваша программа готова!"
  - Превью: 3 рекомендованных упражнения (название + длительность + иконка зоны)
  - Метрика: "14 упражнений • 2 мин/день • Результат за 14 дней"
  - CTA: "Продолжить"

#### 1.9 Reminder Setup
- **Назначение:** Настроить напоминания ДО paywall, получить push-разрешение в контексте
- **Откуда:** Персональный план (1.8)
- **Куда:** Paywall (1.10)
- **Элементы:**
  - Заголовок: "Когда напоминать о растяжке?"
  - Мульти-выбор (toggle): Утро (9:00) / Обед (13:00) / Вечер (18:00) / Каждые 2 часа
  - Переключатель: "Eye break каждые 20 мин"
  - CTA: "Включить напоминания" → системный запрос push
  - Ссылка: "Настроить позже"

#### 1.10 Paywall
- **Назначение:** Конверсия в trial/подписку
- **Откуда:** Reminder Setup (1.9)
- **Куда:** Home (3.1)
- **Элементы:** см. [WIREFRAMES.md](./WIREFRAMES.md) экран 3

### 2. Auth

#### 2.1 Sign In
- **Назначение:** Вход для существующих пользователей
- **Откуда:** Welcome Screen (1.2), Profile (3.4)
- **Куда:** Home (3.1)
- **Элементы:** Email + пароль, Apple Sign In, Google Sign In, "Забыли пароль?", "Создать аккаунт"

#### 2.2 Sign Up
- **Назначение:** Создание аккаунта (отложено — после aha-момента или после первой сессии)
- **Откуда:** Sign In (2.1), после первой сессии (soft prompt)
- **Куда:** Home (3.1)
- **Элементы:** Email + пароль, Apple Sign In, Google Sign In, Terms & Privacy

### 3. Main Tabs

#### 3.1 Home (Tab 1)
- **Назначение:** Центр управления: быстрый доступ к рутинам, стрик, рекомендации
- **Откуда:** Любой таб, session complete, push-уведомление
- **Куда:** Routine Preview (4.1), Eye Exercise (3.3.2), Pain Check-in (5.4), Library (3.2)
- **Элементы:**
  - Streak widget (дни подряд + недельный календарь)
  - Body Map (выбор зоны → переход к рутине)
  - Рекомендованная рутина (персональная, по зоне из профиля)
  - Eye Break quick-action ("Глаза устали? 30 сек")
  - Pain check-in баннер (если не заполнен сегодня)
  - Мотивационная фраза (мягкий тон)

#### 3.2 Library (Tab 2)
- **Назначение:** Каталог всех упражнений с фильтрацией
- **Откуда:** Tab bar, Home (ссылка "Все упражнения")
- **Куда:** Exercise Detail (3.2.1), Routine Preview (4.1)
- **Элементы:**
  - Поиск (по названию)
  - Фильтры: зона тела, длительность, сложность
  - Карточки упражнений (превью, название, длительность, зона, Free/Premium бейдж)
  - Кнопка "Избранное" на каждой карточке

#### 3.2.1 Exercise Detail
- **Назначение:** Подробная информация об упражнении
- **Откуда:** Library (3.2), Routine Preview (4.1)
- **Куда:** Exercise Player (4.2), Paywall (5.1 если locked)
- **Элементы:**
  - Видео-превью (thumbnail)
  - Название, описание, длительность
  - Целевые мышцы, противопоказания, модификации
  - CTA: "Начать" или "Разблокировать" (если premium)
  - Кнопка "Добавить в избранное"

#### 3.3 Programs (Tab 3)
- **Назначение:** Структурированные программы (sciatica, eye)
- **Откуда:** Tab bar, Home (карточка программы)
- **Куда:** Sciatica Program (3.3.1), Eye Program (3.3.2)
- **Элементы:**
  - Карточки программ с прогрессом
  - Sciatica Program (premium бейдж)
  - Eye Program (бесплатно)
  - Описание, количество упражнений, длительность курса

#### 3.3.1 Sciatica Program
- **Назначение:** Структурированная программа для ишиаса с фазами
- **Откуда:** Programs (3.3)
- **Куда:** Symptom Checker (3.3.1.1), Exercise Player (4.2), Paywall (5.1)
- **Элементы:**
  - Disclaimer: "Не заменяет медицинскую консультацию"
  - Symptom Checker вход: "Как болит сегодня?"
  - Фазы: Острая (gentle) / Поддержание (progressive)
  - Список упражнений с прогрессом
  - Red flags screening

#### 3.3.2 Eye Program
- **Назначение:** Упражнения для глаз + 20-20-20 таймер
- **Откуда:** Programs (3.3), Home (Eye Break), Push-уведомление
- **Куда:** Eye exercise (анимация), Home
- **Элементы:**
  - 20-20-20 таймер с визуальной анимацией
  - Список eye exercises (palming, figure-8, focus shifting)
  - Каждое — 20–30 сек с анимированным гидом
  - Работает без звука

#### 3.4 Profile (Tab 4)
- **Назначение:** Профиль, прогресс, настройки
- **Откуда:** Tab bar
- **Куда:** Progress (3.4.1), Pain History (3.4.2), Settings (3.4.3)
- **Элементы:**
  - Аватар, имя (если указано)
  - Subscription status (Free / Pro)
  - Quick stats: всего сессий, минут, текущий стрик
  - Навигация: Прогресс, Боль, Настройки, О приложении

#### 3.4.1 Progress
- **Назначение:** Визуализация прогресса, стрики, бейджи
- **Откуда:** Profile (3.4)
- **Куда:** Share Sheet (5.7)
- **Элементы:**
  - Текущий стрик + лучший стрик
  - Календарь активности (месяц)
  - Бейджи (3, 7, 14, 30 дней)
  - Weekly summary: минуты, сессии, зоны
  - Кнопка "Поделиться прогрессом"

#### 3.4.2 Pain History
- **Назначение:** Визуализация динамики боли
- **Откуда:** Profile (3.4)
- **Куда:** Export (PDF)
- **Элементы:**
  - График боли по зонам (неделя / месяц)
  - Корреляция: боль vs. количество сессий
  - Кнопка "Экспорт для врача" (PDF)

#### 3.4.3 Settings
- **Назначение:** Управление настройками приложения
- **Откуда:** Profile (3.4)
- **Куда:** Подэкраны настроек (3.4.3.1–3.4.3.5)
- **Элементы:**
  - Список секций: Напоминания, Профиль, Подписка, Оформление, Конфиденциальность

### 4. Exercise Flow

#### 4.1 Routine Preview
- **Назначение:** Предпросмотр рутины перед началом
- **Откуда:** Home (3.1) → выбор зоны, Library (3.2), Programs (3.3)
- **Куда:** Exercise Player (4.2), назад к источнику
- **Элементы:**
  - Название рутины, длительность, количество упражнений
  - Список упражнений (порядок, превью, длительность)
  - Целевая зона
  - CTA: "Начать рутину"
  - Кнопка "Назад"

#### 4.2 Exercise Player
- **Назначение:** Воспроизведение видео-упражнения
- **Откуда:** Routine Preview (4.1), Exercise Detail (3.2.1), Push-уведомление
- **Куда:** Следующее упражнение (автопереход) → Session Complete (4.3)
- **Элементы:**
  - Видео (полноэкранное)
  - Play/Pause
  - Прогресс-бар (текущее упражнение)
  - Таймер
  - Текстовые подсказки по технике
  - Кнопки: "Пропустить", "Повторить"
  - Кнопка "×" (выход из рутины)
  - Счётчик: "2 из 3"
  - Аудио toggle (вкл/выкл)

#### 4.3 Session Complete
- **Назначение:** Закрепление привычки, обновление стрика, опциональный pain check-in
- **Откуда:** Exercise Player (4.2) — последнее упражнение
- **Куда:** Home (3.1), Pain Check-in (5.4), Share (5.7)
- **Элементы:**
  - Поздравление: "Отлично! Шея скажет спасибо"
  - Статистика сессии: длительность, упражнения, зоны
  - Стрик обновлён: "День 6!"
  - Pain check-in prompt: "Как ощущения? Оценить?" (опционально)
  - Кнопка "Поделиться" (если milestone)
  - CTA: "На главную"

### 5. Modals / Overlays

#### 5.1 Paywall (повторный)
- **Назначение:** Конверсия при feature-gate
- **Откуда:** Locked content в Library, Programs, Home
- **Куда:** Покупка → разблокировка, или закрытие
- **Элементы:** Аналогично 1.10, но контекстный (упоминание конкретной фичи)

#### 5.2 Mini-Paywall
- **Назначение:** Быстрый inline upsell при нажатии на locked content
- **Откуда:** Карточка упражнения с замком
- **Куда:** Полный Paywall (5.1) или закрытие
- **Элементы:** Компактный блок: "Откройте 50+ упражнений — 7 дней бесплатно", кнопка "Подробнее"

#### 5.3 Push Permission Primer
- **Назначение:** Объяснение ценности push перед системным запросом
- **Откуда:** Reminder Setup (1.9), Settings (3.4.3.1)
- **Куда:** Системный запрос iOS/Android
- **Элементы:** "Напоминания помогают 87% пользователей делать перерывы регулярно", кнопки "Включить" / "Не сейчас"

#### 5.4 Pain Check-in Modal
- **Назначение:** Быстрая оценка боли по зонам
- **Откуда:** Home баннер, Session Complete (4.3)
- **Куда:** Закрытие → данные сохранены
- **Элементы:** Зона боли (из профиля), слайдер 1–10, кнопка "Сохранить"

#### 5.5 Streak Freeze Alert
- **Назначение:** Уведомление об использовании grace day
- **Откуда:** Автоматически при пропуске дня
- **Куда:** Закрытие
- **Элементы:** "Вчера был grace day. Стрик сохранён! Сегодня важно не пропустить"

#### 5.6 Rate App Prompt
- **Назначение:** Запрос рейтинга после позитивного опыта
- **Откуда:** Session Complete (после 3–5 сессии)
- **Куда:** App Store / Google Play рейтинг
- **Элементы:** "Нравится DeskCare?", кнопки "Оценить" / "Позже" / "Нет, спасибо"

#### 5.7 Share Sheet
- **Назначение:** Шеринг достижений
- **Откуда:** Session Complete (milestone), Progress (3.4.1)
- **Куда:** Внешнее приложение (Instagram, WhatsApp, etc.)
- **Элементы:** Автоматическая карточка (бейдж + стрик + ссылка), каналы шеринга

#### 5.8 Milestone Celebration
- **Назначение:** Награда за достижение (3, 7, 14, 30 дней)
- **Откуда:** Session Complete (при достижении milestone)
- **Куда:** Share Sheet (5.7) или Home (3.1)
- **Элементы:** Анимация, бейдж, статистика, кнопка "Поделиться"

### 6. System Screens

#### 6.1 Loading
- **Назначение:** Загрузка данных / синхронизация
- **Откуда:** Любой экран
- **Куда:** Целевой экран
- **Элементы:** Логотип + spinner

#### 6.2 No Internet
- **Назначение:** Информирование об отсутствии соединения
- **Откуда:** Любой экран, требующий сети
- **Куда:** Повтор → целевой экран
- **Элементы:** Иконка, "Нет соединения", "Кешированный контент доступен", кнопка "Повторить"

#### 6.3 Force Update
- **Назначение:** Требование обновить приложение
- **Откуда:** Запуск приложения (если версия критически устарела)
- **Куда:** App Store / Google Play

- **Откуда:** Запуск / любой экран
- **Куда:** Повтор
- **Элементы:** "Технические работы. Скоро вернёмся!", ориентировочное время

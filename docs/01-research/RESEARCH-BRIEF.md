## 1. Elevator Pitch

**DeskCare — приложение для микро-растяжек прямо за рабочим столом.** 2–5 минутные видео-упражнения для шеи, спины, глаз и запястий, специализированные программы для ишиаса и карпального туннеля, умные напоминания — всё, чтобы 80% офисных работников, испытывающих боль от сидячей работы, наконец перестали страдать и начали двигаться между задачами.

## 3. Top-5 Insights — Главные находки

### Insight 1: Android — главное конкурентное преимущество
Wakeout (лидер рынка, Apple App of the Year 2020) — **iOS-only**. 70%+ мирового рынка смартфонов — Android. Кросс-платформенный запуск DeskCare = мгновенный доступ к огромному недообслуженному рынку. Это не фича, это **стратегическое преимущество**.

### Insight 2: Специализированные программы = premium retention
Пользователи с конкретными состояниями (ишиас, карпальный туннель) **в 2–3 раза более мотивированы** платить, чем пользователи "общей растяжки". Алексей (персона с ишиасом) готов платить $7.99/мес без раздумий — потому что боль. Дженнифер (общий дискомфорт) — максимум $4.99/мес после trial. Специализированные программы = **higher ARPU + lower churn**.

### Insight 3: Разрыв "поиск → приложение" — acquisition goldmine
40K–60K/мес ищут "sciatica exercises" в Google. Только 1–3K ищут "sciatica app". Люди **хотят решение, но не знают, что оно может быть в формате приложения**. SEO/ASO-стратегия, таргетирующая exercise-запросы (не app-запросы), может дать огромный органический трафик при минимальных затратах на маркетинг.

### Insight 4: Retention — это THE challenge, а не competition
Конкурентная среда благоприятна. Но wellness-приложения исторически имеют **churn 60–80% в первый месяц**. Все три персоны **уже пробовали и бросили** предыдущие решения. Если DeskCare не решит retention (стрики, умные напоминания, видимый прогресс, ультра-короткий формат) — рынок и позиционирование не спасут.

### Insight 5: Eye exercises — неосвоенная территория
CVS (компьютерный зрительный синдром) поражает 69–94% офисных работников — **больше, чем любая МСК-проблема**. При этом **ни одно** конкурентное приложение не интегрирует eye exercises. 20-20-20 rule + eye yoga — это low-hanging fruit для дифференциации и дополнительный повод открывать приложение каждый день.

## 4. Key Risks — Главные риски

### Риск 1: Low Retention / High Churn
| | |
|---|---|
| **Вероятность** | Высокая |
| **Влияние** | Критическое — убивает unit economics |
| **Митигация** | Стрики с мягкой геймификацией (не Duolingo-guilt); ультра-короткий формат (2–3 мин = "нет excuses"); адаптивные напоминания (не раздражающие, учитывающие DnD/календарь); pain tracking с визуализацией прогресса; социальный элемент (challenges с коллегами). |

### Риск 2: Видеоконтент — высокие upfront costs
| | |
|---|---|
| **Вероятность** | 100% (не риск, а факт) |
| **Влияние** | Среднее — задерживает запуск, увеличивает burn |
| **Митигация** | Начать с 50–80 core routines (не 2,500 как Wakeout). Покрыть: шея (10), спина (10), поясница/ишиас (15), глаза (5), запястья (5), full-body (5). Один инструктор, одна студия, 2–3 дня съёмки. Расширять библиотеку по данным usage. |

### Риск 3: Wakeout расширяется на Android
| | |
|---|---|
| **Вероятность** | Средняя (не анонсировано, но логичный шаг) |
| **Влияние** | Высокое — потеря Android-преимущества |
| **Митигация** | Двигаться быстро. Первый качественный продукт в нише получает SEO/ASO-позиции, которые сложно отобрать. Дифференцироваться через специализированные программы (ишиас) — то, что Wakeout не делает и вряд ли будет. |

### Риск 4: Юридический / Регуляторный
| | |
|---|---|
| **Вероятность** | Низкая (при правильном подходе) |
| **Влияние** | Высокое — штрафы, переклассификация, иски |
| **Митигация** | Advisory board с первого дня. Юридический review всех маркетинговых материалов. Disclaimers. Противопоказания для каждого упражнения. Red flags screening. Professional liability insurance. Не использовать слова "лечение", "терапия", "диагностика". |

### Риск 5: Конкуренция с бесплатными YouTube-видео
| | |
|---|---|
| **Вероятность** | Средняя |
| **Влияние** | Среднее — ограничивает conversion |
| **Митигация** | YouTube не даёт: напоминания, таргетинг по частям тела, прогресс-трекинг, pain tracking, стрики, персонализацию, оффлайн-доступ, структурированные программы. Convenience + система = moat. |

## 7. Verdict — Финальный вердикт

### ✅ GO

**Условия:**

1. **Кросс-платформенный запуск** (iOS + Android) — без этого теряется ключевое преимущество
2. **Профессиональный видеоконтент** — минимум 50 упражнений к запуску, снятых с real инструктором
3. **Advisory board** (хотя бы 1 PT + 1 врач) до публичного запуска
4. **Retention-first дизайн** — стрики, умные напоминания, ультра-короткий формат — не afterthought, а core product

### Почему GO, а не CAUTIOUS GO

- Рынок огромный ($6–9.5B TAM) и быстро растущий (+12–18% CAGR)
- Конкурентная среда практически пустая в нашей нише
- Проблема затрагивает 80%+ целевой аудитории
- Temporal window открыто: Android пуст, культурный тренд на нашей стороне, FDA облегчила путь
- Технический стек стандартный — основной барьер не в коде, а в контенте
- Два revenue stream (mass market desk stretches + premium specialized programs) диверсифицируют риск

### Единственный scenario для NO-GO

Если невозможно обеспечить **качественный видеоконтент** (бюджет, инструктор, PT oversight) — проект не имеет смысла. Без профессионального видео DeskCare не будет отличаться от существующих низкокачественных приложений. Видео — это **the product**, а не feature.

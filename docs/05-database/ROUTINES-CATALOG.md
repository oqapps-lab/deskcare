# DeskCare — Каталог рутин (формулы сборки)

**Дата:** 2026-04-25
**Источник:** `EXERCISES-LIBRARY-SUPABASE.md` (атомы) + `DATABASE-SCHEMA.md` (таблицы `routines`, `routine_exercises`).
**Назначение:** Концертный список рутин для seed-скрипта Supabase. Каждая рутина — упорядоченная последовательность пар `(exercise.code, reps)`. Длительность вычисляется как `Σ (atom.duration_seconds × reps + rest_seconds)`.

---

## Принцип формулы

Запись `N1×4` означает: атом с `code='N1'` (длительность 6 сек) проигрывается **бесшовно 4 раза подряд**. Это даёт 24 секунды и 4 «реальных» цикла наклонов головы (внутри одного атома уже один полный цикл «вправо-центр-влево»).

Формула рутины: **`A×n + B×m + C×k = total_sec`**.

Полная формула с паузами: `A×n + (rest 2s) + B×m + ...` — но в MVP `rest_seconds=0` для всех item-ов. Перерывы закрывает сам бесшовный стык атомов.

**Важно при подборе reps:**
- `mobility` / `strength` — обычно **2–4 повтора** (нужно «прогреть»).
- `stretch` — **1–2 повтора** (удержание уже встроено в атом).
- `nerve_glide` — **1–2 повтора**, не больше (раздражает нерв).
- `breathing` — **1–3 цикла** (S6 = 14 сек уже глубокий).
- Атом «по 1 на сторону» — лучше **чётное число reps** для симметрии.

**Коды кросс-категорий** (B12, F11–F15 и т.д.) в seed-скрипте подменяются исходным `code` (N9, N7, N5, N8, N9, B10) — это один и тот же атом в БД. Но в этом каталоге везде используется код, соответствующий **активной категории рутины**, чтобы было визуально понятно.

# 10. Пример seed-скрипта (фрагмент)

```sql
-- Создаём атом N5 один раз
INSERT INTO exercises (code, slug, title, title_en, description, video_url,
                       duration_seconds, reps_inside_atom, difficulty, exercise_type)
VALUES ('N5', 'shoulder-rolls', 'Круги плечами', 'Shoulder Rolls',
        'Большой круг плечами вперёд и назад. Снимает напряжение трапеций.',
        'exercise-videos/shoulder-rolls/video.mp4',
        10, 'по 2 в каждую сторону', 1, 'mobility');

-- Связываем с двумя зонами (M:N)
INSERT INTO exercise_body_zones (exercise_id, body_zone_id, is_primary)
SELECT e.id, bz.id, (bz.slug = 'neck')
FROM exercises e CROSS JOIN body_zones bz
WHERE e.code = 'N5' AND bz.slug IN ('neck', 'full_body');

-- Создаём рутину R2
INSERT INTO routines (slug, title, body_zone_id, duration_seconds, routine_type)
SELECT 'neck-quick-2min', 'Шея · Быстрая', bz.id, 116, 'zone_based'
FROM body_zones bz WHERE bz.slug = 'neck';

-- Заполняем routine_exercises с reps
WITH r AS (SELECT id FROM routines WHERE slug = 'neck-quick-2min'),
     items(sort_order, code, reps) AS (VALUES
       (1, 'N5', 2),
       (2, 'N1', 4),
       (3, 'N2', 4),
       (4, 'N3', 2),
       (5, 'N6', 1),
       (6, 'N9', 3),
       (7, 'N10', 1)
     )
INSERT INTO routine_exercises (routine_id, exercise_id, sort_order, reps)
SELECT r.id, e.id, items.sort_order, items.reps
FROM items, r, exercises e
WHERE e.code = items.code;
```

Полный seed-скрипт для всех 20 рутин — в `deskcare/supabase/seed/routines.sql` (создаётся отдельной задачей).

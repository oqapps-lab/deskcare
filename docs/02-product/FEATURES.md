## MVP (Must Have) — Запуск

### F1: Onboarding Quiz

| Поле | Описание |
|------|----------|
| **Название** | Onboarding Quiz |
| **User Story** | Как новый пользователь, я хочу ответить на 4–5 вопросов о своих зонах дискомфорта и типе работы, чтобы приложение сразу подобрало релевантные упражнения |
| **Acceptance Criteria** | ☐ 4–5 экранов: тип работы (офис/удалёнка/гибрид), зоны боли (мульти-выбор: шея, спина, поясница, глаза, запястья), интенсивность боли (1–10), частота перерывов, цель (снять боль / профилактика / энергия) ☐ Результат сохраняется в профиле ☐ Можно изменить в настройках ☐ Пропуск возможен (default profile) ☐ Время прохождения < 60 секунд |
| **Экран** | Onboarding flow (4–5 шагов) |
| **Приоритет** | P0 |
| **Сложность** | M |

### F2: Body-Part Targeting

| Поле | Описание |
|------|----------|
| **Название** | Body-Part Targeting (Таргетинг по зоне боли) |
| **User Story** | Как пользователь с болью в шее, я хочу нажать "Шея" и получить целевую 3-минутную рутину, чтобы снять напряжение именно в этой зоне |
| **Acceptance Criteria** | ☐ Главный экран: визуальный выбор зоны (шея, спина/поясница, глаза, запястья, полное тело) ☐ Нажатие на зону → экран с рутиной (2–5 мин) ☐ Рутина подобрана по onboarding-профилю и истории ☐ Quick-start: одно нажатие от главного экрана до видео ☐ Минимум 8–10 упражнений на каждую зону |
| **Экран** | Home Screen → Routine Screen |
| **Приоритет** | P0 |
| **Сложность** | L |

### F3: Видео-плеер с упражнениями

| Поле | Описание |
|------|----------|
| **Название** | Exercise Video Player |
| **User Story** | Как пользователь, я хочу видеть профессиональное видео с инструктором, который показывает упражнение с двух ракурсов, чтобы правильно выполнить движение |
| **Acceptance Criteria** | ☐ Видео-плеер: play/pause, прогресс-бар, таймер ☐ Текстовые подсказки по технике поверх видео ☐ Аудио-инструкции (можно отключить) ☐ Информация: целевые мышцы, противопоказания, модификации ☐ Кнопки: "Пропустить упражнение", "Повторить" ☐ Поддержка оффлайн (кешированные видео) ☐ Автопереход к следующему упражнению в рутине |
| **Экран** | Exercise Player Screen |
| **Приоритет** | P0 |
| **Сложность** | XL |

### F4: Библиотека упражнений

| Поле | Описание |
|------|----------|
| **Название** | Exercise Library |
| **User Story** | Как пользователь, я хочу просмотреть все доступные упражнения с фильтрацией по зоне тела и длительности, чтобы найти конкретное упражнение или собрать свою рутину |
| **Acceptance Criteria** | ☐ Каталог: 50–80 упражнений к запуску ☐ Фильтры: зона тела, длительность (20с / 1мин / 2мин / 5мин), сложность (лёгкая / средняя) ☐ Каждое упражнение: превью, описание, целевые мышцы, противопоказания ☐ Поиск по названию ☐ Free vs. Premium бейдж ☐ Избранное (сохранить) |
| **Экран** | Library Screen (таб) |
| **Приоритет** | P1 |
| **Сложность** | L |

### F5: Smart Reminders

| Поле | Описание |
|------|----------|
| **Название** | Smart Reminders (Умные напоминания) |
| **User Story** | Как удалённый работник, который забывает делать перерывы, я хочу настраиваемые напоминания с конкретными упражнениями, чтобы регулярно двигаться, не раздражаясь |
| **Acceptance Criteria** | ☐ Настройка расписания: рабочие часы (начало/конец), интервал (каждые 30/45/60/90 мин), дни недели ☐ DnD-aware: пауза в определённые часы ☐ Push-уведомление с конкретным упражнением ("Растяжка шеи — 2 мин") ☐ Нажатие на уведомление → сразу в плеер ☐ Можно отложить на 15/30 мин ☐ Eye reminders отдельно: каждые 20 мин (правило 20-20-20) ☐ Настройка тона: мягкий / нейтральный / мотивирующий |
| **Экран** | Settings → Reminders |
| **Приоритет** | P0 |
| **Сложность** | L |

### F6: Eye Exercise Module

| Поле | Описание |
|------|----------|
| **Название** | Eye Exercises (Упражнения для глаз) |
| **User Story** | Как офисный работник с усталостью глаз, я хочу быстрые 20–30 секундные упражнения для глаз с напоминанием каждые 20 минут, чтобы снять зрительное напряжение |
| **Acceptance Criteria** | ☐ 20-20-20 правило с таймером и визуальной анимацией ☐ 5+ eye exercises: palming, figure-8, focus shifting, blinking, eye circles ☐ Каждое — 20–30 сек с анимированным гидом ☐ Отдельная секция на главном экране ☐ Напоминание каждые 20 мин (настраиваемо) ☐ Работает без звука (анимация + текст) |
| **Экран** | Home → Eye Section; Eye Exercise Screen |
| **Приоритет** | P0 |
| **Сложность** | M |

### F7: Базовая Sciatica-программа

| Поле | Описание |
|------|----------|
| **Название** | Sciatica Program (Базовая) |
| **User Story** | Как человек с ишиасом, я хочу структурированную программу desk-friendly упражнений с учётом фазы боли, чтобы безопасно облегчить симптомы без физиотерапевта |
| **Acceptance Criteria** | ☐ 10–15 упражнений, все desk-friendly (сидя/стоя) ☐ 2 фазы: острая (gentle) + поддержание (progressive) ☐ Symptom checker на входе: "как болит сегодня?" → подбор фазы ☐ Противопоказания для каждого упражнения ☐ Red flags screening: при опасных симптомах → "обратитесь к врачу" ☐ Disclaimer: "не заменяет медицинскую консультацию" ☐ Premium content (платная подписка) |
| **Экран** | Programs → Sciatica Program |
| **Приоритет** | P1 |
| **Сложность** | XL |

### F8: Стрики и базовая геймификация

| Поле | Описание |
|------|----------|
| **Название** | Streaks & Gamification |
| **User Story** | Как пользователь, который бросает приложения через неделю, я хочу видеть свою серию активных дней и получать мотивацию продолжать, чтобы не потерять привычку |
| **Acceptance Criteria** | ☐ Стрик-каунтер: дни подряд с хотя бы одной сессией ☐ Недельный календарь активности на главном экране ☐ Milestone badges: 3 дня, 7 дней, 14 дней, 30 дней ☐ Мягкий тон: "Ты сделал 5 дней подряд!" (не Duolingo-guilt) ☐ Восстановление стрика: 1 пропуск не обнуляет (grace day) ☐ Weekly summary: минут растяжки, зоны, прогресс |
| **Экран** | Home Screen (стрик-виджет), Profile → Progress |
| **Приоритет** | P1 |
| **Сложность** | M |

### F9: Pain Tracking

| Поле | Описание |
|------|----------|
| **Название** | Pain Tracking (Трекер боли) |
| **User Story** | Как пользователь с хронической болью, я хочу отслеживать уровень боли по зонам каждый день, чтобы видеть, помогают ли упражнения |
| **Acceptance Criteria** | ☐ Ежедневный check-in: зона боли + шкала 1–10 (простой слайдер) ☐ График боли по времени (неделя, месяц) ☐ Корреляция: боль vs. количество сессий ☐ Опционально (не навязывается) ☐ Экспорт данных (для врача/PT) ☐ Субъективная оценка (не медицинская диагностика) |
| **Экран** | Home → Pain Check-in; Profile → Pain History |
| **Приоритет** | P1 |
| **Сложность** | M |

## Won't Have (сейчас)

| Фича | Почему НЕ делаем |
|------|-----------------|
| **Real-time posture monitoring** | Требует постоянную камеру, battery drain, privacy concerns. Сложность не оправдана на MVP. |
| **Телемедицина / консультации PT** | Переводит продукт в категорию telehealth — другая регуляторика, лицензирование, страхование. |
| **Диагностика состояний** | FDA перевёдёт в medical device. Юридически опасно. Используем symptom-based подбор. |
| **Интеграция с EMR** | B2B фича, требует HIPAA compliance, не нужна для consumer MVP. |
| **Hardware (sensorы, wearable)** | Капиталоёмко, другой бизнес. Интегрируемся с existing (Apple Watch) в v2. |
| **Meal planning / nutrition** | Не наш фокус. Размываем позиционирование. Конкурируем с MyFitnessPal — бессмысленно. |
| **Медитация / mindfulness** | Конкурируем с Headspace/Calm ($2B+ рынок). Наш фокус — физическое движение, не ментальное. |
| **Marketplace инструкторов** | Создаёт проблемы качества контента, модерации, юридической ответственности. Контент курируем сами. |

## MVP Scope Summary

### Экраны

| # | Экран | Фичи |
|---|-------|-------|
| 1 | **Onboarding (4–5 шагов)** | F1 |
| 2 | **Auth (Sign Up / Sign In)** | F10 |
| 3 | **Home** | F2 (body-part selector), F6 (eye section), F8 (streak widget), F9 (pain check-in) |
| 4 | **Routine Preview** | F2 (список упражнений в рутине) |
| 5 | **Exercise Player** | F3 (видео-плеер) |
| 6 | **Library** | F4 (каталог упражнений) |
| 7 | **Programs** | F7 (sciatica program) |
| 8 | **Eye Exercise** | F6 (20-20-20 + eye yoga) |
| 9 | **Pain Tracking** | F9 (check-in + history) |
| 10 | **Progress** | F8 (стрики, badges), F9 (pain graph) |
| 11 | **Profile** | F10 (настройки, подписка, профиль) |
| 12 | **Paywall** | Подписка (Pro / Annual) |

### Сводка

| Метрика | Значение |
|---------|----------|
| **Фичи MVP** | 10 (F1–F10) |
| **Экраны** | ~12 уникальных экранов |
| **Приоритет P0 (блокер запуска)** | 5 фич: Onboarding, Body-Part Targeting, Video Player, Smart Reminders, Eye Exercises |
| **Приоритет P1 (нужно к запуску)** | 5 фич: Library, Sciatica Program, Streaks, Pain Tracking, Profile |
| **Видеоконтент к запуску** | 50–80 упражнений |
| **Оценка сложности** | 2 XL + 4 L + 3 M = ~3–4 месяца разработки (1 developer) |

## 1. Проблема

### Какая боль?

**Офисные и удалённые работники испытывают хроническую физическую боль от сидячей работы — шея, спина, глаза, запястья — но не могут систематически решить эту проблему на рабочем месте.**

Это не абстрактный "дискомфорт". Это:

| Конкретная боль | Данные |
|-----------------|--------|
| Шея затекает так, что больно поворачивать голову | 58.6% офисных работников ([Nature, 2025](https://www.nature.com/articles/s41598-025-30155-6)) |
| Поясница стреляет после 2+ часов сидения | 52.5% офисных работников |
| Глаза горят и расплываются к 15:00 | 69–94% работающих за экранами ([Nature meta-analysis, 2023](https://www.nature.com/articles/s41598-023-28750-6)) |
| Запястья ноют к концу недели | 3–6% популяции, выше у интенсивных пользователей клавиатуры |
| Ишиас — стреляющая боль вниз по ноге | до 40% людей столкнутся хотя бы раз, 13M новых эпизодов/год (США) |

### Насколько она острая? (Frequency × Severity)

```
                 ВЫСОКАЯ ЧАСТОТА
                       ▲
                       │
        Боль в шее ●   │   ● Зрительное напряжение
        (ежедневно,    │   (ежедневно, средняя
         средняя)      │    тяжесть)
                       │
                       │
        Боль в         │   ● Боль в запястьях
        пояснице ●     │   (3–5 дней/нед,
        (ежедневно,    │    средняя)
         средне-       │
         высокая)      │
────────────────────────┼──────────────────────►
  НИЗКАЯ ТЯЖЕСТЬ       │              ВЫСОКАЯ ТЯЖЕСТЬ
                       │
                       │   ● Ишиас
                       │   (эпизодически,
                       │    очень высокая)
                       │
                 НИЗКАЯ ЧАСТОТА
```

**Оценка:** Проблема **высокочастотная** (каждый рабочий день) и **средне-высокой тяжести** (влияет на работу, сон, качество жизни). Для ишиаса — эпизодическая, но **критической** тяжести.

### Как люди решают проблему сейчас?

| Решение | % использующих | Стоимость | Удовлетворённость |
|---------|---------------|-----------|-------------------|
| **Ничего не делают** | ~40% | $0 | ❌ Боль усиливается |
| **YouTube видео** | ~25% | $0 | ⚠️ Нет системы, противоречивые советы |
| **Эргономичные аксессуары** | ~20% | $40–$400 | ⚠️ Помогают частично, не решают |
| **Физиотерапия** | ~10% | $80–$320/мес | ✅ Работает, но дорого и долго |
| **Обезболивающие** | ~15% | $10–$30/мес | ❌ Маскирует, не решает |
| **Wellness-приложения** | ~5% | $0–$13/мес | ⚠️ Бросают через 1–3 недели |
| **Desktop break reminders** | ~5% | $0–$5 | ⚠️ Только напоминание, нет контента |

### Почему текущие решения не работают?

| Решение | Почему не работает |
|---------|-------------------|
| **YouTube** | Нет системы. Каждый раз ищешь новое видео. 10 мин на поиск 5-мин упражнения. Противоречивые советы (для ишиаса — опасно). Нет прогресса. Нет напоминаний. |
| **Wakeout (iOS)** | Нет Android (70%+ рынка). Широкий фокус (спальня, кухня, диван). Нет специализированных программ (ишиас). Анимации вместо видео. Агрессивный paywall. |
| **Desktop reminders (Stretchly)** | Только текст "потянитесь". Нет видео, нет конкретных упражнений. Прерывает flow state. Десктоп-only. |
| **Физиотерапия** | $40 co-pay × 2/нед = $320/мес. 1 час + дорога. Бросают через 3–6 визитов. Упражнения на бумажке теряются. |
| **Обезболивающие** | Маскируют симптом. Побочные эффекты при длительном приёме. Не решают причину. |
| **Kaia Health / Hinge Health** | Только через работодателя (B2B). Сессии 10–15 мин (слишком длинно для перерыва). Не desk-specific. |

## 2. Наше решение

### Ключевое обещание

**DeskCare показывает тебе конкретное видео-упражнение для твоей боли — за 2–3 минуты, прямо за рабочим столом, без коврика и переодевания.**

### Value Proposition

> **Для** удалённых и офисных работников, **у которых** болит шея, спина, глаза или запястья от сидячей работы, **наш** DeskCare **позволяет** снять боль и предотвратить ухудшение через 2–5 минутные видео-упражнения прямо за столом, **в отличие от** YouTube (нет системы), Wakeout (iOS-only, нет специализации) и физиотерапии (дорого, долго), **потому что** мы единственные, кто объединяет: кросс-платформенность + профессиональное видео + таргетинг по зоне боли + специализированные программы (ишиас) + упражнения для глаз — всё desk-friendly.

### 3 главных дифференциатора

| # | Дифференциатор | vs. Конкуренты | Источник |
|---|----------------|----------------|----------|
| 1 | **Кросс-платформа (iOS + Android)** | Wakeout — iOS-only. Moova — слабый. Stretchly — десктоп. Android (70%+ рынка) — практически пуст. | [COMPETITORS.md](../01-research/COMPETITORS.md) — Saturation Score: 3/10 |
| 2 | **Body-part targeting + специализированные программы** | Ни один конкурент не даёт "болит шея → вот 3-мин рутина для шеи". Ни один B2C-app не имеет структурированной sciatica-программы с фазами. | [COMPETITORS.md](../01-research/COMPETITORS.md) — Gap Analysis |
| 3 | **Eye exercises (20-20-20 + eye yoga)** | CVS поражает 69–94% офисных работников — БОЛЬШЕ чем любая МСК-проблема. Ни один конкурент не интегрирует eye exercises. Полная пустота. | [COMPETITORS.md](../01-research/COMPETITORS.md) — Gap #3; [DOMAIN-RESEARCH.md](../01-research/DOMAIN-RESEARCH.md) |

### "Aha-moment"

**Момент, когда пользователь понимает ценность DeskCare:**

> Пользователь открывает приложение, нажимает "Шея" (или "Глаза"), получает 3-минутную видео-рутину, делает её прямо за столом — и **чувствует реальное облегчение через 3 минуты**.

**Ключевые элементы Aha-moment:**
1. **Таргетинг** — "Оно знает, что у меня болит именно шея" (не generic stretch)
2. **Скорость** — "Это заняло 3 минуты, я даже не встала со стула"
3. **Эффект** — "Шея реально стала лучше"
4. **Видео** — "Живой инструктор показывает мне что делать, а не анимация"

**Time to Aha:** < 3 минуты (первая сессия с body-part targeting)

## 3. Validation Checklist

### Проблема подтверждена данными

- [x] **80.81%** офисных работников испытывают МСК-боли ([Nature, 2025](https://www.nature.com/articles/s41598-025-30155-6))
- [x] **58.6%** — боль в шее, **52.5%** — боль в пояснице
- [x] **69–94%** — компьютерный зрительный синдром ([Nature, 2023](https://www.nature.com/articles/s41598-023-28750-6))
- [x] **13M** новых эпизодов ишиаса/год в США
- [x] **70%** удалённых работников сообщают о новых/ухудшившихся МСК-симптомах ([Frontiers, 2023](https://www.frontiersin.org/journals/public-health/articles/10.3389/fpubh.2023.1155745/full))
- [x] Микро-перерывы снижают дискомфорт на **27–48%** без снижения продуктивности (NIOSH)

### Аудитория достаточно большая

- [x] TAM: **$6.0–9.5B** (workplace wellness + sciatica treatment + digital MSK)
- [x] SAM: **$900M–$1.4B** (англоязычные рынки, smartphone users)
- [x] SOM Year 3: **$5–12M ARR** (50K–80K платящих)
- [x] **1B+** офисных работников в мире (Forrester)
- [x] **32.6M** работают удалённо в США (BLS)
- [x] Digital MSK рынок растёт +18% CAGR

### Конкурентов можно обойти

- [x] Market Saturation Score: **3/10** — "blue ocean" ([COMPETITORS.md](../01-research/COMPETITORS.md))
- [x] Wakeout (лидер) — **iOS-only**, нет специализированных программ, нет eye exercises
- [x] Android — **практически пуст**. Moova — единственный прямой конкурент, слабый (⭐4.0)
- [x] Kaia Health ушла из B2C (поглощена Sword Health, янв. 2026) — конкурентов стало меньше
- [x] Ни один конкурент не объединяет: кросс-платформа + видео + desk-specific + ишиас + глаза
- [x] DeskCare занимает **верхний правый квадрант** positioning map — уникальная позиция

### Техническая реализация возможна

- [x] Стек стандартный: Expo SDK 55 + React Native + TypeScript + Supabase + Adapty
- [x] Основная сложность — **видеоконтент**, не код ([DOMAIN-RESEARCH.md](../01-research/DOMAIN-RESEARCH.md))
- [x] FDA: General Wellness Product — **не требуется** одобрение как медицинское устройство
- [x] HIPAA: **не требуется** для consumer wellness app
- [x] AI-фичи (form correction) — Фаза 2+, не MVP
- [x] Единственный hard requirement: профессиональный видеоконтент (50–80 упражнений к запуску)

## 4. Риски, которые нужно решить

| Риск | Уровень | Митигация в продукте |
|------|---------|---------------------|
| **Low retention** (60–80% churn в первый месяц — норма для wellness) | 🔴 Критический | Стрики, pain tracking, ультра-короткий формат (2–3 мин = "нет отговорок"), smart reminders, visible progress |
| **YouTube как бесплатная альтернатива** | 🟡 Средний | Convenience moat: таргетинг + напоминания + прогресс + система. YouTube не даёт ничего из этого. |
| **Качество видеоконтента** | 🔴 Критический | Минимум 50 профессиональных видео к запуску. Один инструктор, одна студия, 2–3 дня съёмки. |
| **Юридический (ишиас-программы)** | 🟡 Средний | Advisory board (PT + MD) с первого дня. Disclaimers. Symptom-based подбор, не diagnosis-based. |
